# OMOS Platform Console WordPress Plugin

Production WordPress plugin scaffold for OMOS.OneGodian.com.

## Purpose

This plugin turns OMOS from a normal WordPress website into a controlled platform console.

It supports three operating surfaces:

1. Public-facing website
2. Logged-in user console
3. Administrative control panel

## Production Rule

If a feature is not implemented, versioned, documented, repeatable, logged where applicable, and testable, it is not operational in the current version.

## Module Map

- Dashboard
- Page Installer
- Mega Menu
- Artifacts
- Tools
- Dashboard Builder
- Registry / Verify
- Docs
- Shop Integration
- Compliance
- Import / Export
- System Health
- Settings

## Runtime Integration

WordPress is the presentation, distribution, commerce, and user-console layer.

Node remains the authority, runtime, API, versioning, and execution layer for OMOS.OneGodian.org.
