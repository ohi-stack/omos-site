## WordPress Implementation Notes

- Use Mega Menu plugin/grid layout where each parent item supports 4-column dropdowns.
- Use the 6-column OMOS Tools layout only where the theme/plugin supports full-width dropdowns.
- Keep `/tools/` as the master landing page and route all tool pages beneath it.
- Keep legal/governance pages in the footer and in OMOS mega menu only where context requires.
- Confirm all pages exist before menu activation; placeholders should be published as draft pages or hidden noindex pages until content is final.
