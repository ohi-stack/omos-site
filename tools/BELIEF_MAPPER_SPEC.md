# OMOS Belief Mapper Specification

Version: v1.0.0
Shortcode: `[omos_belief_mapper]`
Recommended page: `/tools/belief-mapper/`

## Purpose

The Belief Mapper is the entry tool for helping a visitor identify their OneGodian journey stage. It should function as an educational and identity-reflection tool, not as a coercive conversion engine.

## Journey Stages

```text
Seeker      — exploring the One God concept
Believer    — affirming One God as primary truth
OneGodian   — actively identifying with the framework
Elder       — long-term contributor and guide
```

## Recommended Question Dimensions

- Ontology: What does the user believe about ultimate reality?
- Unity: Does the user believe the source of existence is singular or multiple?
- Relationship: How does the user understand their relationship to that source?
- Tradition: What tradition, if any, informs current belief?
- Identity: How does the user currently identify spiritually or philosophically?
- Community: What role does community play in their belief practice?
- Purpose: Does the user believe their life has purpose from a higher source?

## Output

The tool should return:

- journey stage result
- short explanation
- recommended next page or product
- optional declaration generator CTA
- optional membership or newsletter CTA

## Production Requirements

- avoid pressure language
- clearly state that the tool is educational
- do not claim legal classification from a short questionnaire
- do not determine protected status automatically
- do not store sensitive data unless explicitly disclosed

## Future Integrations

- personalized content recommendations
- declaration generator handoff
- membership stage suggestions
- QRV profile creation only after explicit user consent

## Compliance Note

The Belief Mapper does not convert, certify, diagnose, or legally classify a user. It provides a guided reflective result based on voluntary user input.
